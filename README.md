'Voyagear' is a e-commerce website focused on selling travel and hiking gears. It's a frontend based project created with React.JS and Tailwind CSS. Its a great project for understanding state management, CRED operations, dynamic routing etc.

Stack used:
- React JS (for scalable and interactive frontend)
- Tailwind CSS (For clean and responsive UI)
- JSON server (for mock api to pass dynamic data)
- Axios (for advanced and clean http requests)

Functionalities:
- Auth context
- Wishlist & Cart
- Buy now option
- Search & sorting products
- Checkout page
- Order history
